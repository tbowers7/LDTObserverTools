NEO Confirmation Page Ephemeris Generator

Status: In development

Overview

  • neocp_ephem: An ephemeris-generating tool for querying the JPL Scout database for short-shelf- life NEOs that have not yet been assigned a Horizons identifier. This tool will turn the returned data into a file that can be ingested into the LDT TCS for observations.

Usage

The script usage can be displayed by calling the script with the -h option:

$ neocp_ephem -h
usage: neocp_ephem [-h] obj_id

Generate LDT Ephemeris Files for NEOCP objects

positional arguments:
  obj_id      The NEOCP temporary designation ID (e.g., 'P10vY9r')

options:
  -h, --help  show this help message and exit
