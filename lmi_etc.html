LMI Exposure Time Calculator

Status: In development

Overview

  • lmi_etc: Python GUI version of the LMI exposure time calculator (http://www2.lowell.edu/users/massey/LMI/etc_calc.php).

Usage

The script usage can be displayed by calling the script with the -h option:

$ lmi_etc -h
usage: lmi_etc [-h]

LMI Exposure Time Calculator
